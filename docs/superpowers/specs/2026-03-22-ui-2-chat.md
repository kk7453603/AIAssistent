# UI Spec 2: Chat UI

## Goal
Full-featured chat interface that streams responses from PAA API, renders markdown, shows tool execution in real-time, and supports collapsible `<think>` blocks.

## Dependencies
- Spec 1 (scaffold) must be complete

## Components

### ChatPage (`src/pages/ChatPage.tsx`)
Main chat view. Contains MessageList, InputBar, ConversationSidebar.

### MessageList (`src/components/chat/MessageList.tsx`)
Scrollable list of messages. Auto-scrolls to bottom on new messages. Each message rendered by MessageBubble.

### MessageBubble (`src/components/chat/MessageBubble.tsx`)
Renders a single message:
- **User messages**: right-aligned, colored background
- **Assistant messages**: left-aligned, markdown rendered
- **`<think>` blocks**: collapsible section with "Thinking..." header. Closed by default. Shows tool calls with status icons (spinner → checkmark/cross)
- **Tool status**: during streaming, shows "🔍 knowledge_search..." → "✓ knowledge_search" in think block
- **Markdown**: rendered via `react-markdown` + `remark-gfm` + syntax highlighting (`rehype-highlight`)
- **Code blocks**: copy button, syntax highlighting
- **Sources**: if answer contains sources, show as collapsible citation list

### InputBar (`src/components/chat/InputBar.tsx`)
- Textarea with auto-resize (shift+enter for newline, enter to send)
- Send button
- Stop button (visible during streaming, aborts fetch)
- File upload button (future: drag-and-drop)
- Model selector dropdown (fetches from `/v1/models`)

### ConversationSidebar (`src/components/chat/ConversationSidebar.tsx`)
- List of conversations (stored in localStorage or Tauri fs)
- New conversation button
- Delete conversation

## SSE Streaming Implementation

```typescript
// src/hooks/useChat.ts
export function useChat(apiUrl: string) {
  const [messages, setMessages] = useState<ChatMessage[]>([]);
  const [isStreaming, setIsStreaming] = useState(false);
  const [toolStatus, setToolStatus] = useState<ToolStatusDelta[]>([]);
  const abortRef = useRef<AbortController | null>(null);

  const sendMessage = async (content: string, conversationId: string) => {
    const userMsg = { role: 'user', content };
    setMessages(prev => [...prev, userMsg]);
    setIsStreaming(true);
    setToolStatus([]);

    abortRef.current = new AbortController();

    const response = await fetch(`${apiUrl}/v1/chat/completions`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      signal: abortRef.current.signal,
      body: JSON.stringify({
        model: selectedModel,
        messages: [...messages, userMsg],
        stream: true,
        metadata: { user_id: 'desktop', conversation_id: conversationId },
      }),
    });

    const reader = response.body!.getReader();
    const decoder = new TextDecoder();
    let assistantContent = '';

    while (true) {
      const { done, value } = await reader.read();
      if (done) break;

      const chunk = decoder.decode(value);
      for (const line of chunk.split('\n')) {
        if (!line.startsWith('data: ')) continue;
        const data = line.slice(6);
        if (data === '[DONE]') break;

        const parsed = JSON.parse(data);
        const delta = parsed.choices?.[0]?.delta;

        // Tool status events
        if (delta?.tool_status) {
          const ts = JSON.parse(delta.tool_status);
          setToolStatus(prev => [...prev, ts]);
          continue;
        }

        // Content tokens
        if (delta?.content) {
          assistantContent += delta.content;
          setMessages(prev => {
            const updated = [...prev];
            const last = updated[updated.length - 1];
            if (last?.role === 'assistant') {
              last.content = assistantContent;
            } else {
              updated.push({ role: 'assistant', content: assistantContent });
            }
            return updated;
          });
        }
      }
    }
    setIsStreaming(false);
  };

  const stopStreaming = () => abortRef.current?.abort();

  return { messages, isStreaming, toolStatus, sendMessage, stopStreaming };
}
```

## Think Block Parsing

PAA agent wraps reasoning in `<think>...</think>` tags. Parse and render:

```typescript
// src/utils/parseThinkBlocks.ts
export function parseThinkBlocks(content: string): { thinking: string; answer: string } {
  const match = content.match(/<think>([\s\S]*?)<\/think>\s*([\s\S]*)/);
  if (!match) return { thinking: '', answer: content };
  return { thinking: match[1].trim(), answer: match[2].trim() };
}
```

Render thinking as collapsible `<details>` with tool status icons.
